# 01 – Oops that was private (secret exposure)

**Educational use only.** This server is intentionally vulnerable for workshop purposes.

## Challenge

A flight-booking server with search, details, booking status, airport info, and promos. Something valuable is tucked away - if you know where to look.

## Goal

Find the flag. Poke around; the tools have more to say than they let on.

## Flag format

`CTF{...}` (CTF prefix with content inside).

## How to run

Configure your MCP client to run this command via STDIO, then use the tools to find the flag. For step-by-step setup with MCPJam, see the [break README](../README.md).
